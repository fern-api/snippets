The Snippets API uses API keys to authenticate requests. Your API key will have the prefix `fern_`. Your API keys are private, so be sure to keep them secure. Don't share your secret API keys in publicly accessible areas such as GitHub, client-side code, and so forth.

<CodeBlocks>
    <CodeBlock title="cURL">
    ```bash
    curl -X POST \
        --url "https://api.buildwithfern.com/snippets" \
        --header "X-Fern-API-Key: fern_moN3KI_1AseQp1UI5U9ysYFsuNo4P3Il" \
    ```
    </CodeBlock>
    
To get an API Key, please [contact us](mailto:support@buildwithfern.com).
